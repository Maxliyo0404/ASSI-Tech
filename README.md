# ASSI-Tech - Raqamli Transformatsiya Xizmatlari

## 🚀 Loyiha Tavsifi

ASSI-Tech - O'zbekistonning yetakchi IT va raqamli transformatsiya xizmatlari kompaniyasi uchun yaratilgan zamonaviy web sayt. 

Loyihada Next.js, React, Tailwind CSS va zamonaviy web texnologiyalari ishlatilgan.

## ✨ Xususiyatlar

- ✅ Responsive dizayn (barcha qurilmalarda ishlaydi)
- ✅ Modern UI/UX
- ✅ Smooth animatsiyalar
- ✅ SEO optimizatsiya
- ✅ Tez yuklash vaqti
- ✅ O'zbekcha interfeys

## 📋 Komponentlar

1. **Header** - Navigatsiya meni va logotip
2. **Hero** - Bosh qism animatsiyalari bilan
3. **Product/Services** - Xizmatlar ro'yxati
4. **About** - Kompaniya haqida ma'lumot
5. **Footer** - Pastki bo'lim va bog'lanish ma'lumotlari

## 🛠️ Texnologiyalar

- **Next.js 16.2.10** - React framework
- **React 19.2.4** - UI kutubxonasi
- **Tailwind CSS 4** - CSS framework
- **PostCSS** - CSS processor

## 📦 O'rnatish

```bash
# Loyihani klonlash
git clone https://github.com/Maxliyo0404/ASSI-Tech.git

# Papkaga o'tish
cd ASSI-Tech

# Dependency o'rnatish
npm install

# Development serverni ishga tushirish
npm run dev

# Browser'da ochish
# http://localhost:3000
```

## 🏗️ Build

```bash
# Production build
npm run build

# Production serverni ishga tushirish
npm start
```
